Popis projektu:
Generátor a ověřovač hesel
Konzolová aplikace v jazyce C, která umožňuje uživateli generovat bezpečná hesla na základě zvolených parametrů:
(délka, použití velkých/malých písmen, číslic a symbolů).
Součástí je kontrola síly hesla podle pěti kritérií:
(unikátnost znaků, absence opakování, délka, kompromitace a počet parametrů).
Program menu zobrazuje aktuální parametry, heslo a sílu hesla.


Pokyny pro spuštění:
Naklonovat do VisualStudio. Spustit Projekt_VBA.sln


Spuštění jednotkových testů:
Dostat se do složky projektu přesněji do ../src
A zadat příkaz make.
Následně make run.
